JakeClaw — GPTv1 rough draft build

This folder contains a working "rough draft" implementation of the JakeClaw project generated by the assistant (GPTv1). It mirrors the planned project structure and includes scripts, templates, skill definitions, configs, docs, and CI skeletons.

Purpose
- Provide a concrete, runnable (where possible) skeleton for JakeClaw.
- Mark places where user inputs (keys, credentials, hostnames) are required in `GPTv1/needs/`.

How to use
1. Inspect `GPTv1/needs/` for values you must supply before running scripts.
2. Review and adapt scripts under `GPTv1/scripts/` to your environment before executing.
3. Run `bash GPTv1/scripts/deploy-test.sh` after a real deployment to validate.

Notes
- This is a draft scaffold for comparison/iteration with other agent outputs (e.g., Claude Code).
- Do not run scripts as-is in production; update secrets and confirm all placeholders first.
