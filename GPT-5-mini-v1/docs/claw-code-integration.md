# Claw-code Integration (Draft)

This document explains how the `claw-code` Rust harness integrates with OpenClaw in JakeClaw.

Overview:
- `claw-code` is built from source and provides a CLI (`claw`) and an optional Axum server.
- We expose `claw` tools into OpenClaw as SKILL.md wrappers generated by `scripts/expose-claw-tools.sh`.

Integration pattern:
1. Build `claw-code`: `cd claw-code/rust && cargo build --release`.
2. Symlink binary: `ln -sf $(pwd)/target/release/claw /usr/local/bin/claw`.
3. Run `scripts/expose-claw-tools.sh` to generate dynamic skill wrappers under `skills/claw-tools-dynamic/`.
4. Ensure `config/openclaw-config.json` includes the directory in `skills.load.extraDirs`.

Notes:
- If `claw` exposes a JSON manifest, `expose-claw-tools.sh` will parse it and create SKILL.md files. The manifest format varies by version — adjust parsing accordingly.
- For API-mode, you may prefer to start the Claw Axum server and route an OpenAI-compatible proxy (LiteLLM) to it.
